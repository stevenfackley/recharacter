# ReCharacter

Self-help web app that helps U.S. veterans upgrade less-than-honorable discharges through
mental-health / liberal-consideration petitions — routing to the correct review board and form,
building the nexus argument, drafting the supporting statement, and exporting a ready-to-file
packet the veteran owns and submits themselves.

**Posture:** document assembly + information, never legal advice.

**Status:** design phase. See [`docs/superpowers/specs/2026-07-05-recharacter-design.md`](docs/superpowers/specs/2026-07-05-recharacter-design.md).
